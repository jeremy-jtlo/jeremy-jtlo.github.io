Portfolio Style Guide

Interactive Elements

Action Button

CSS selector:

.button

Sample code:

<a href="#" class="button">Default Button</a>

Rendered element:

Default Button

In-text link

CSS selector:

.text-link

Sample code:

<a href="#" class="text-link">Text Link</a>

Rendered element:

Text Link

In-text link (Dark)

CSS selector:

.text-link.dark

Sample code:

<a href="#" class="text-link dark">Text Link (Dark)</a>

Rendered element:

Text Link (Dark)

Image Link

CSS selector:

.image-link

Sample code:

  <img src="img/failtol1.png">
</a>

Text Elements

Headings

CSS selector:

.header

Sample code:

<h2 class="header">Header</h2>

Rendered element:

Header

Underlined Header

CSS selector:

.header.underlined

Sample code:

<h2 class="header underlined">Underlined</h2>

Rendered element:

Underlined
