problem

Opening a New Tab page in my browsers often showed a grid of things that the browser assumes I want to see. I'd rather have something that fits my needs a little bit more closely, since I frequent sites with many branching subsections, while also using web apps like Gmail.

approach

Make one! I've always wanted an excuse to try out a front-end framework, and I know a bit of JS already. There were some features that would be interesting to implement without too much trouble: some time-sensitive greetings, a live clock widget, and cascading link menus.
